API Documentation Page


Web app title: kickFlix

This achievement's project is a server-side, web application with the purpose of being a martial arts movie library. Hence the name "kickFlix"! This web app will provide users with information about different movies, actors, directors, and sub genres (within the martial arts genre). Additionally, users will be able to sign-up, update their personal information, and create a list of their favorite martial arts movies.


Movie Endpoints

Description Method Endpoint Query Parameters Request Body (example) Response
Returns a list of all movies GET /movies None None Array of movie objects with info.
Returns movie info. by title GET /movies/:title title (movie title) None Movie object with title, genre, director, starring, year, imageURL, featured
Returns genre description by it's name GET /genre/:name name (genre name) None Genre object with name and description
Returns director info. by name GET /directors/:name name (director's name) None Director object with name, bio, and birth year

User Endpoints

Description Method Endpoint Query Parameters Request Body (example) Response
Create a new user POST /users None Format: JSON
Example:
{
"name": "Lucy",
"favoriteMovie": ["The Forbidden Kingdom"]
} 		Created user object, or error message
Update user's info. PUT /users/:id username (user's id) {
"id": 1,
"name": "Larry Updated Name",
"favoriteMovie": []
} 		Updated user object info.
Add movie to user's favorites POST /users/:id/:movieTitle username (id and movie title) None Message: "Movie has been added to user's array."
Remove a movie from user's favorites DELETE /users/:id/:movieTitle username (id and movie title) None Message: "Movie has been removed from user's array."
De-register a user DELETE /users/:id username None Message: "User has been deleted."

Entity and Attributes

Entity Attributes
Movies ID, Title, Description, Genre, Director, URL to image, Featured (Y/N)
Director ID, Name, Bio, Birth year, Death year
Genre ID, Name, Description
Users ID, Username, Password, Email, Birthday, Favorite movies
