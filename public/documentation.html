API Documentation Page


Web app title: kickFlix

This achievement's project is a server-side, web application with the purpose of being a martial arts movie library. Hence the name "kickFlix"! This web app will provide users with information about different movies, actors, directors, and sub genres (within the martial arts genre). Additionally, users will be able to sign-up, update their personal information, and create a list of their favorite martial arts movies.


Movie Endpoints

Description Method Endpoint Query Parameters Request Body (example) Response
Returns a list of all movies GET /movies None None Array of movie objects with info.
Returns movie info. by title GET /movies/:title title (movie title) None Movie object with title, genre, director, starring, year, imageURL, featured
Returns genre description by it's name GET /genre/:name name (genre name) None Genre object with name and description
Returns director info. by name GET /director/:name name (director's name) None Director object with name, bio, and birth year

Genre Entity and Attributes

genreID Name Description
1 Action Movies in the action genre are defined by risk and stakes. While many movies may feature an action sequence, to be appropriately categorized inside the action genre, the bulk of the content must be action-oriented, including fight scenes, stunts, car chases, and general danger.
2 Comdey The comedy genre is defined by events that are intended to make someone laugh, no matter if the story is macabre, droll, or zany. Comedy can be found in most movies, but if the majority of the film is intended to be a comedy you may safely place it in this genre. The best comedy movies range throughout this entire spectrum of humor.
3 Animated The animation genre is defined by inanimate objects being manipulated to appear as though they are living. This can be done in many different ways and can incorporate any other genre and sub-genre.
4 Parody A parody mocks and specifically targets a single piece of art or connected body of work. A parody is more precise, and more limited.
